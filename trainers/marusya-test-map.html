Маруся: один понятный путь

Убрали лишнее: оставили одну линию с маркерами, убрали округление из маршрута. Идём коротко: отрицательные числа → знаки → уравнения. Геометрия — только как мягкий резерв или смена деятельности.

Не даём выбирать из хаосаИдём сверху вниз, по одному шагу.
Один блок = 5–7 минутЛучше остановиться раньше, чем довести до усталости.
Фиксируем наблюденияГде угадывает, где читает, где просит подсказку.
