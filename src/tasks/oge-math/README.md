# ОГЭ Математика — методические карты заданий

Эта папка предназначена для методической структуры открытого банка тренажёров.

## Назначение

Для каждого задания ОГЭ хранится:

- список подтипов;
- правила генерации;
- правила автоматической проверки;
- типичные ошибки;
- подсказки;
- требования к интерфейсу;
- связь с Notion и будущими тренажёрами.

## Статусы
