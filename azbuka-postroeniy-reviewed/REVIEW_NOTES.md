# REVIEW_NOTES — UX-доработка тренажёров

Основа: шесть HTML‑тренажёров после доработки Claude + дополнительная проверка и зачистка несоответствий текста/логики.

## Исправлено сквозно

- Убрана подпись «версия для тестирования» из ученического интерфейса.
- Нижняя панель подсказок и инструментов сделана липкой на мобильном экране.
- Из бейджей убраны технические единицы длины; вместо `AB = … px` используется учебная формулировка.
- Проверена синтаксическая валидность JavaScript во всех шести файлах.
- Проверено прохождение правильного сценария в браузере: все шесть тренажёров доходят до `done=true` без JS‑ошибок.

## По тренажёрам

### Отрезок, равный данному

- Клик по `O` сразу строит окружность сохранённым раствором `AB`.
- Текст шага и текущая подсказка синхронизированы: больше нет требования «подтвердить раствор».

### Угол, равный данному

- Радиус дуги у `A` выбирается учеником; точки `B`, `C`, `D`, `E` пересчитываются динамически.
- Добавлены зоны «ДАННЫЙ УГОЛ» и «СТРОИМ НОВЫЙ УГОЛ».
- Точка `E` ярко подсвечивается при засечке.

### Биссектриса угла

- Добавлен фантом точки `C` и дуга‑призрак из `B`, чтобы ученик видел, где встретятся внутренние дуги.
- Сообщение об ошибке радиуса сделано менее техническим.

### Середина отрезка

- Окружность из `A` строится сразу после выбора раствора `AB`; окружность из `B` строится одним кликом по `B`.
- Точку `O` можно отметить любым инструментом.
- Тексты шагов обновлены под новую механику.

### Перпендикуляр через точку на прямой

- Убраны скрытые заранее заданные `A/B`: ученик кликает по прямой `a`, точка `A` ставится в месте проекции клика, `B` появляется симметрично относительно `M`.
- Фантомные `A/B` показываются до клика.
- Начальная подсказка обновлена под новую механику.

### Перпендикуляр из внешней точки

- Радиус окружности из `M` свободный: принимается любой корректный радиус, пересекающий прямую `a` в двух точках.
- Под курсором показываются фантомная окружность и будущие точки `A/B`.
- Малый радиус отклоняется с понятной причиной.
- Исправлена опечатка цвета в превью (`COL.red` → `COL.r`).

## Осторожность

Это функционально‑UX проверка «глазами ученика» и smoke test правильного сценария. Полная методическая экспертиза формулировок доказательств, возрастной доступности языка и строгой геометрической корректности может быть отдельным этапом.
