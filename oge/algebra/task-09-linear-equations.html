Задание 9: Уравнения и их системы

ОГЭ по математике • Алгебра • Средняя сложность

Условие задачи (из ФИПИ)

Решите уравнение:

5x + 4 = 3x - 8

Решение

Шаг 1. Переносим слагаемые с x в одну сторону, числа — в другую:

5x - 3x = -8 - 4

Шаг 2. Упрощаем:

2x = -12

Шаг 3. Делим обе части на коэффициент при x:

x = -12 ÷ 2

x = -6

Ответ: -6

⚠️ Типичные ошибки

  • Ошибка знаков: При переносе через "=" знак меняется на противоположный
  • Потеря минуса: -8 - 4 = -12, а не -4
  • Деление: -12 ÷ 2 = -6, а не 6

Проверка решения

Подставим найденное значение в исходное уравнение:

5·(-6) + 4 = 3·(-6) - 8

-30 + 4 = -18 - 8

-26 = -26 ✓

Равенство верное, значит решение правильное.
