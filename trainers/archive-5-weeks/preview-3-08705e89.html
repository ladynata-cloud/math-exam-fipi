Тренажёр истинных и ложных утверждений

Формат: ученик проверяет геометрическое утверждение, ищет ловушку, вспоминает теорему или строит контрпример. Можно дать как домашку после урока.

Что внутри
  • экзаменационный режим: 3 утверждения → ответ цифрами;
  • карточки “верно/неверно” с мгновенным разбором;
  • темы: треугольники, окружность, четырёхугольники, углы, площади;
  • учительский режим: банк утверждений, поиск, экспорт/импорт.

Экзаменационный режим

Укажите номера верных утверждений. Ответ записывается без пробелов, например: 13.

Карточки: верно или неверно?

Лучший режим для закрепления после урока: одно утверждение — один быстрый выбор — короткая причина.

Тема 0 / 0

Тренировка по темам

Можно назначать точечно: например, только окружность или только ловушки про параллелограммы.

Ошибки ученика

Здесь собираются утверждения, в которых ученик ошибался. Очень удобно для “добить слабое место”.

Учительский режим

Банк утверждений можно расширять: добавляй формулировки из архива, свои ловушки и короткие объяснения.

Мини-алгоритм 19-й

1. Найди ключевые слова: “любой”, “каждый”, “всегда”, “существует”.
2. Вспомни точную теорему, а не “вроде похоже”.
3. Для сомнительного утверждения ищи контрпример: прямоугольник не ромб, ромб не квадрат, равные площади не дают равенства фигур.
4. В ответ пиши только номера верных утверждений.
