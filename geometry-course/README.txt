Геометрия · 7 класс — интерактивные тренажёры
==============================================

70 одностраничных тренажёров (Главы I–IV учебника Атанасяна) + страница-оглавление.
Чистый HTML + Canvas, без библиотек и без сервера — выкладывается как статика.

СТРУКТУРА (вложенность не менять):
  index.html        — главная: карта курса и прогресс
  trainers/*.html   — сами тренажёры

КАК ВЫЛОЖИТЬ:
  Загрузите всю папку geometry-course как есть на любой статический хостинг.
  Папка trainers/ должна лежать рядом с index.html (на уровень ниже).
  Ссылки внутри (открытие тренажёра, кнопка «к списку») — относительные, работают сами.

ПРОГРЕСС:
  Зелёные галочки хранятся в браузере ученика (localStorage), на его устройстве.
  Никуда не отправляется. Сброс — очистка данных сайта в браузере.

РЕЖИМ УЧИТЕЛЯ (кнопка справа вверху на главной):
  - метки типов и фильтр: Уроки / Практика / Зачёты / Построения;
  - отметь тренажёры галочкой -> внизу кнопка "Скопировать ссылки" даёт готовый
    список с прямыми URL — вставь в дневник/чат/LMS как набор на урок;
  - показываются коды тренажёров (ch4-p2-t8 и т.п.).
  Режим запоминается в этом браузере и на учеников не влияет (у них по умолчанию выкл).

СБОР ПРОГРЕССА КЛАССА (teacher.html):
  - Ученик на главной жмёт "Сдать прогресс", вводит имя -> получает короткий код,
    копирует и присылает учителю (чат/форма). Код ~80 символов, влезает в сообщение.
  - Учитель открывает teacher.html, вставляет коды (по одному в строке, можно пачкой)
    -> таблица класса: сколько пройдено у каждого, по главам, кто буксует. Экспорт CSV.
    Повторный код от ученика обновляет его строку. Данные класса хранятся в браузере учителя.
  - Необязательно: автосбор в Google-таблицу (Apps Script) — инструкция в самой teacher.html.
