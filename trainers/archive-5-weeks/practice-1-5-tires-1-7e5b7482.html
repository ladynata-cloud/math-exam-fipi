Шины · маркировка и геометрия колеса

Сегодня учимся решать задачи на шины. Сначала разберём маркировку 195/60 R16, потом научимся считать высоту боковины, диаметр колеса и сравнивать колёса по пробегу.

Шаг 1

Как читать маркировку шины

У каждой автомобильной шины есть маркировка из трёх частей. Разберём заводскую маркировку.

195 / 60 R 16
маркировка автомобильной шины
Первое число: 195. Что оно обозначает?
Второе число: 60. Что оно обозначает?
Запись R16. Что обозначает 16?
Буква R в маркировке. Что она обозначает?
Главное
B = 195 — ширина шины в миллиметрах
p = 60 — процент: высота боковины = 60% от ширины
R — радиальная конструкция (буква без числа)
d = 16 — диаметр диска в дюймах (1 дюйм = 25,4 мм)
Внимание: R — это не радиус! Радиус колеса считается просто: R = D / 2 (половина диаметра).
Шаг 2

Собери таблицу параметров

Из маркировки 195/60 R16 можно вычислить все характеристики колеса. Заполни таблицу по порядку — каждый столбец проверяется отдельно.

Маркировка B, мм p, % d, дюйм d, мм H, мм D, мм
195/60 R16
формулы: d_мм = d · 25,4  ·  H = B · p / 100  ·  D = 2H + d_мм
Как заполнять: B — первое число (мм). p — второе число (просто число процента, без знака %). d — число после R (в дюймах). d, мм — переведи дюймы в миллиметры (× 25,4). H — найди по формуле H = B · p / 100. D — соедини: D = 2H + d_мм.
Образец заполнения
B = 195 мм
p = 60 (%)
d = 16 дюймов
d, мм = 16 · 25,4 = 406,4 мм
H = 195 · 60 / 100 = 117 мм
D = 2 · 117 + 406,4 = 640,4 мм
Заводское колесо: D = 640,4 мм
D = H + dмм + H = 2H + dмм
наверху боковина H, диск d, снизу ещё боковина H
образец

Шаблон итоговой таблицы

Дальше в задачах 4 и 5 ты будешь заполнять такую таблицу для двух шин сразу. Возвращайся сюда, если запутаешься в формулах.

Маркировка B, мм p, % d, дюйм d, ммd · 25,4 H, ммB · p / 100 D, мм2H + d_мм
195/60 R16 195 60 16 406,4 117 640,4
Порядок заполнения слева направо: маркировка → B → p → d (дюйм) → d (мм) → H → D. Сначала таблица, потом сравнения.

Задача 2

Сравнение радиусов колёс

На сколько миллиметров радиус колеса с шиной 185/65 R16 больше, чем радиус колеса с шиной 215/55 R16?

Алгоритм (как во всех задачах на шины):
1) заполняем таблицу параметров для обеих шин  ·  2) находим разность диаметров  ·  3) если спрашивают про радиус — делим на 2 (ведь R = D/2).
Маркировка B, мм p, % d, дюйм d, ммd · 25,4 H, ммB · p / 100 D, мм2H + d_мм
185/65 R16
215/55 R16
Заполни обе строки слева направо. Только когда таблица собрана — переходим к разности диаметров и радиусов.
Подсказки: 185/65 R16 → d_мм = 16·25,4 = 406,4; H = 185·65/100 = 120,25; D = 2·120,25 + 406,4 = 646,9.  ·  215/55 R16 → d_мм = 406,4; H = 215·55/100 = 118,25; D = 2·118,25 + 406,4 = 642,9.
1Разность диаметров
D1 − D2 = 646,9 − 642,9 = ?
мм
646,9 − 642,9 = 4 мм. Это разница ДИАМЕТРОВ. Но в задаче спросили про радиусы — будет ещё один шаг.
2Что делать дальше?
Мы нашли разницу диаметров — 4 мм. А в задаче спрашивают про радиусы. Что нужно сделать?
Радиус — это половина диаметра (R = D / 2). Если диаметры отличаются на 4 мм, то радиусы отличаются ровно в 2 раза меньше: 4 : 2 = 2 мм.
3Разность радиусов
R1 − R2 = (D1 − D2) / 2 = 4 / 2 = ?
мм
4 : 2 = 2 мм. Это окончательный ответ.
Решение задачи 2 — единый табличный метод
Шаг 1 — итоговая таблица параметров:
185/65 R16 → B=185, p=65, d=16″, d_мм=406,4, H=120,25, D₁ = 646,9 мм
215/55 R16 → B=215, p=55, d=16″, d_мм=406,4, H=118,25, D₂ = 642,9 мм
Шаг 2 — разность диаметров: D1 − D2 = 646,9 − 642,9 = 4 мм
Шаг 3 — переход к радиусам: R = D / 2, поэтому R1 − R2 = (D1 − D2) / 2 = 4 / 2 = 2 мм
Ответ: 2
Методический комментарий: один алгоритм на все задачи на шины — маркировка → таблица параметров → диаметр → дальше (разность, радиус, проценты). Отдельная формула радиуса R = H + d_мм/2 верна математически, но запоминать её незачем: достаточно знать D = 2H + d_мм и R = D/2.

Главная ошибка: ученик находит разницу диаметров 4 и сразу пишет ответ 4. Нужно помнить, что радиус — это половина диаметра, поэтому ответ — 2 мм.
Задача 3

Диаметр заводского колеса

Найдите диаметр колеса автомобиля, выходящего с завода. Заводская маркировка — 195/60 R16. Ответ — в миллиметрах.

D = 2H + dмм  ·  H = B · p / 100  ·  dмм = d · 25,4
1Параметры из маркировки
B = ? мм
мм
2Высота боковины H
H = B · p / 100 = 195 · 60 / 100 = ?
мм
195 · 60 = 11700, делим на 100 → 117 мм.
3Диск в миллиметрах
dмм = d · 25,4 = 16 · 25,4 = ?
мм
16 · 25,4 = 406,4 мм.
4Диаметр колеса
D = 2H + dмм = 2 · 117 + 406,4 = ?
мм
2 · 117 = 234, потом 234 + 406,4 = 640,4 мм.
Решение задачи 3
B = 195 мм, p = 60, d = 16 дюймов
H = 195 · 0,6 = 117 мм
dмм = 16 · 25,4 = 406,4 мм
D = 2 · 117 + 406,4 = 234 + 406,4 = 640,4 мм

Запомни: D — это полный диаметр колеса. В нём две боковины (сверху и снизу), поэтому именно 2H, а не H.
Задача 4

Замена шин: изменение диаметра

На сколько миллиметров уменьшится диаметр колеса, если заменить заводские шины (195/60 R16) шинами 205/50 R17?

Алгоритм: сначала заполняем итоговую таблицу для обеих шин — слева направо, по столбцам. Только после этого считаем разность диаметров.
Маркировка B, мм p, % d, дюйм d, ммd · 25,4 H, ммB · p / 100 D, мм2H + d_мм
195/60 R16 195 60 16 406,4 117 640,4
205/50 R17
Сверху — заводское колесо (его параметры уже посчитаны на шаге 2). Снизу — новое: заполни строку слева направо.
Подсказка для строки 205/50 R17: B=205, p=50, d=17″. d_мм = 17·25,4 = 431,8. H = 205·50/100 = 102,5. D = 2·102,5 + 431,8 = 636,8.
?Что произошло с диаметром?
По таблице: D₁ = 640,4 мм, D₂ = 636,8 мм. Диаметр увеличился или уменьшился?
=На сколько уменьшится?
|D₁ − D₂| = 640,4 − 636,8 = ?
мм
640,4 − 636,8 = 3,6 мм. В ответе пишем положительное число (модуль).
Решение задачи 4
Шаг 1 — итоговая таблица параметров:
195/60 R16 → B=195, p=60, d=16″, d_мм=406,4, H=117, D₁ = 640,4 мм
205/50 R17 → B=205, p=50, d=17″, d_мм=431,8, H=102,5, D₂ = 636,8 мм
Шаг 2 — сравнение: 640,4 > 636,8 → диаметр уменьшился
Шаг 3 — разность: |D₁ − D₂| = 640,4 − 636,8 = 3,6 мм
Ответ: 3,6
Методический комментарий: сначала систематически заполняем таблицу для обеих шин — это страхует от путаницы «какой H для какой шины». Только после того, как таблица собрана, делаем сравнение и считаем разность. Типичные ошибки: (1) считать сразу разность, не строя таблицу; (2) забыли модуль и написали −3,6; (3) перепутали столбцы.
Задача 5

Пробег за оборот и проценты

На сколько процентов увеличится пробег автомобиля при одном обороте колеса, если заменить заводские шины (195/60 R16) шинами 215/45 R18? Результат округли до десятых.

наглядно: пробег = длина окружности = π · D
?Главное наблюдение
Почему в задаче 5 можно сравнивать диаметры, а не считать длины окружностей полностью?
Шаг A · параметры

Сначала — таблица параметров для обеих шин. Старая строка уже заполнена; заполни новую (215/45 R18) слева направо.

Маркировка B, мм p, % d, дюйм d, ммd · 25,4 H, ммB · p / 100 D, мм2H + d_мм
195/60 R16 195 60 16 406,4 117 640,4
215/45 R18
Параметры строки 215/45 R18: B=215, p=45, d=18″, d_мм=18·25,4, H=215·45/100, D=2H+d_мм.
Подсказка для строки 215/45 R18: d_мм = 18·25,4 = 457,2. H = 215·45/100 = 96,75. D = 2·96,75 + 457,2 = 193,5 + 457,2 = 650,7.
Шаг Б · проценты (таблица 2×2)

Из таблицы параметров берём D₁ = 640,4 (старое) и D₂ = 650,7 (новое). За 100% принимаем то, с чем сравниваем — старое заводское колесо.

что сравниваем?натуральная величинапроцент
новое колесо (D₂)650,7 ммx %
старое колесо (D₁)640,4 мм100 %
D₁ · x = D₂ · 100  ⇒  x = D₂ · 100 / D₁ пропорция «крест-накрест»
Найди x = 650,7 · 100 / 640,4 = ? (округли до десятых)
%
650,7 · 100 = 65070; 65070 / 640,4 ≈ 101,6. Это процент НОВОГО относительно старого.
=Это процент или «на сколько»?
x = 101,6% — это вся величина нового относительно старого. А «на сколько процентов увеличится» = ?
% (ответ)
101,6 − 100 = 1,6 %. «101,6%» — это размер нового пробега относительно старого; вопрос — на сколько процентов увеличилось, поэтому вычитаем 100%.
Решение задачи 5
Шаг A — таблица параметров:
195/60 R16 → B=195, p=60, d=16″, d_мм=406,4, H=117, D₁ = 640,4 мм
215/45 R18 → B=215, p=45, d=18″, d_мм=457,2, H=96,75, D₂ = 650,7 мм
Шаг Б — почему π сокращается: один оборот = длина окружности C = π·D. Отношение пробегов: C₂/C₁ = π·D₂/π·D₁ = D₂/D₁.
Шаг В — пропорция: новое — x %, старое — 100 %. Из таблицы 2×2: x = D₂·100/D₁ = 650,7·100/640,4 ≈ 101,6 %.
Шаг Г — финал: 101,6 % — это размер нового относительно старого. «На сколько увеличится» = 101,6 − 100 = 1,6 %.
Ответ: 1,6
Методический комментарий: ученики часто пишут в ответе 101,6 — это значит, что они не различают «процент от» и «на сколько процентов». Финальный шаг с вычитанием 100 — обязательный. Также частая ошибка — считать длины окружностей в полном виде с π, не заметив, что π сокращается. Анимация и mini-quiz нужны именно для этого вывода.

Главные формулы

H = B · p / 100 высота боковины из процента
D = 2H + dмм диаметр всего колеса
dмм = d · 25,4 дюймы → мм
R = D / 2 радиус — половина диаметра

Пробег за оборот

C = π · D длина окружности = пробег за 1 оборот

При сравнении двух колёс π сокращается — поэтому работаем только с диаметрами.
