Шины · таблица параметров

Центром тренажёра является одна большая таблица. Все вычисления — по одному алгоритму, для всех шин сразу.

Этап 1

Таблица из условия задачи

В каждой задаче на шины сначала идёт таблица допустимых маркировок. По ней мы только выбираем шины: строка — ширина, столбец — диск, ячейка — профиль. Полные расчёты пойдут позже, на этапе 3.

Не путать с расчётной таблицей!
Таблица условия → читаем, что разрешено.
Расчётная таблица → считаем d, H, D, R (это будет этап 3).
Этап 2

Читаем маркировку шины

Маркировка устроена однотипно: B/p R r. Прежде чем строить таблицу, научимся уверенно вытаскивать три числа из любой маркировки.

Запомнить:
B — ширина шины в миллиметрах (первое число).
p — профиль в процентах (второе число, между / и R). Это процент от ширины, а НЕ миллиметры.
R — тип конструкции (радиальная), не радиус.
r — диаметр диска в дюймах (число после R). 1 дюйм = 25,4 мм.
Этап 3 · центр тренажёра

Расчётная таблица параметров

Алгоритм для всех шин одинаковый: B → p → r → d → H → D → R. Заполняем таблицу по столбцам — сначала B для всех шин, потом p для всех, и т.д. Так лучше запоминается единый порядок действий.

Этап 5

Сравнение пробега 2×2

Когда меняем шины, пробег за один оборот = длина окружности C = π·D. Но π сокращается, поэтому достаточно сравнить диаметры. Дальше — классическая таблица 2×2.

Округление в ответах

«Авто» — стандартно для типовых задач: проценты до десятых, разности диаметров до десятых, ширина в мм до целого.

Подсказки

В режиме «Учитель» подсказки видны всегда. Здесь — для режимов «Обучение» и «Тренировка».

База заданий (JSON)

Импортированный набор сохраняется в localStorage и переживёт перезагрузку страницы. Формат — JSON одного набора (как в экспорте).

Все доступные наборы

Встроенные наборы удалить нельзя. Пользовательские — можно. Если текущий набор был удалён или его id больше не найден, тренажёр автоматически откроет набор по умолчанию.
