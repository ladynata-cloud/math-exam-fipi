Что нужно, чтобы решить задание 9

В ответе — одно число. Если корней два, а в условии сказано «запишите меньший (больший) из корней» — выбираем нужный. Дробный ответ записываем десятичной дробью через запятую: 1,5  (а не 3/2).

Три главных промаха: потерять знак при переносе, потерять корень x = 0 и записать «меньший» корень вместо «большего» у отрицательных чисел. Дальше — как их не совершить.

Линейные уравнения

Главное правило. Слагаемое, которое «перепрыгивает» через знак =, меняет знак: было + — станет −.

План на 4 шага

1.Раскрой скобки (множитель — на каждое слагаемое; минус перед скобкой меняет все знаки внутри).
2.Перенеси: всё с x — влево, числа — вправо (со сменой знака).
3.Приведи подобные: отдельно иксы, отдельно числа.
4.Раздели обе части на коэффициент при x.

Если коэффициенты дробные — сначала умножь всё уравнение на НОК знаменателей, дроби исчезнут.

Отчаянный метод (если запутался): всё, что с x, — влево; всё без x — вправо. Всегда. Потом привёл подобные и разделил. Цена — иногда возишься с лишними знаками, зато не собьёшься.

Квадратные уравнения

ax² + bx + c = 0.  Сначала смотрим на вид — половина задач решается без дискриминанта.

c = 0 ax² + bx = 0 → вынеси x: x(ax + b) = 0. Корни: x = 0 и скобка = 0. Не теряй ноль!
b = 0 ax² + c = 0 → вырази x², возьми ±√. Корня два: + и −.
полное через дискриминант.
D = b² − 4ac,   x = −b ± D2a

Если D > 0 — два корня; D = 0 — один; D < 0 — корней нет (в задании 9 такого не будет).

Теорема Виета — «для быстрых»

Если a = 1:  x₁ + x₂ = −p,  x₁ · x₂ = q. Подбираешь два числа с нужной суммой и произведением.

Честная цена Виета. Работает быстро, только когда a = 1 и корни целые. Не подобрал за полминуты — не мучайся, иди через дискриминант, он не подведёт никогда.
«Меньший / больший корень»: расставь корни на прямой. У отрицательных −5 меньше, чем −2. Больший — тот, что правее.

Дробно-рациональные уравнения

Есть x в знаменателе — обязательна проверка ОДЗ: знаменатель не может быть нулём. Корень, обнуляющий знаменатель, — посторонний, его выбрасывают.

дробь = дробь пропорция → умножь «крест-накрест».
дробь = 0 числитель = 0, при этом знаменатель ≠ 0.
x и дробь x + ax = b → умножь всё на x, получишь квадратное.
две дроби приведи к общему знаменателю → квадратное → проверь ОДЗ.

Если под дробью встречается замена (например одинаковые скобки) — можно ввести t и упростить. Но на ОГЭ почти всегда хватает общего знаменателя.

Как заполнять бланк

Ответ — число без пробелов и лишних знаков. Минус пишем, если корень отрицательный. Дробный ответ — десятичной дробью через запятую: 0,5  −2,5  1,75. Целый ответ — просто число: 10.

Внутри тренажёра при ответе можно писать и 3/2, и 1,5, и 1.5 — засчитается; но в настоящем бланке нужна запятая.
