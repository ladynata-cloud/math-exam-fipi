# GeoGebra Offline (Variant A)

## Что такое Variant A
Variant A — это самохостинг GeoGebra Math Apps Bundle вместе с локальной codebase. Такой подход полностью исключает сетевые запросы к geogebra.org и гарантирует работу оффлайн на mathexam.space и на GitHub Pages (включая подпапку репозитория).

## Куда положить GeoGebra Math Apps Bundle
Скопируйте распакованный бандл в корень репозитория:

```
GeoGebra/deployggb.js
GeoGebra/HTML5/5.0/web3d/...
```

## Куда положить файл GeoGebra
Положите файл апплета в:

```
ggb/15.ggb
```

## Демо-страница
Готовая страница демо:

```
/ggb/15/
```

Открывать можно как локально, так и на GitHub Pages с подпапкой репозитория (например `/math-exam-fipi/ggb/15/`).

## Проверка оффлайн-режима
1. Откройте DevTools → Network.
2. Введите фильтр `geogebra.org`.
3. Должно быть 0 запросов.

## Типичные ошибки
- **Неправильный путь к бандлу** — проверьте, что `GeoGebra/` лежит в корне и пути относительные.
- **Забыли завершающий слэш** в `setHTML5Codebase("../../GeoGebra/HTML5/5.0/web3d/")`.
- **404 на `GeoGebra/...`** — не загружен бандл или перепутана структура папок.
- **CSP блокирует ресурсы** — разрешите загрузку локальных `script`, `font`, `wasm`.
