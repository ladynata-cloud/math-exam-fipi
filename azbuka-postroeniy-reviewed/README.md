# Азбука построений — обновлённая линейка тренажёров

Статический модуль для сайта: единая страница `index.html` запускает шесть HTML‑тренажёров в маршруте, хранит локальный прогресс, позволяет назначать задания и экспортировать отчёт ученика.

## Что внутри

- `index.html` — единая линейка с прогрессом и заданиями.
- `manifest.json` — список тренажёров и порядок прохождения.
- `trainers/` — шесть доработанных тренажёров.
- `CODEX_PROMPT.md` — промпт для Codex/разработчика.
- `DEPLOYMENT_CHECKLIST.md` — короткий чеклист выкладки.
- `REVIEW_NOTES.md` — что было проверено и что изменено после UX‑разбора.

## Тренажёры

1. Отрезок, равный данному — `trainers/segment-trainer-v6.html`
2. Угол, равный данному — `trainers/angle-trainer-v2.html`
3. Биссектриса угла — `trainers/bisector-trainer-v4.html`
4. Середина отрезка — `trainers/midpoint-trainer-v2.html`
5. Перпендикуляр через точку на прямой — `trainers/perpendicular-trainer-v2.html`
6. Перпендикуляр из точки вне прямой — `trainers/perpendicular-external-trainer-v2.html`

## Выкладка

Положите папку `azbuka-postroeniy-reviewed/` в публичную директорию сайта и откройте:

```text
/azbuka-postroeniy-reviewed/index.html
```

Все файлы статические, внешних зависимостей нет. Важно сохранять структуру папок, потому что `index.html` открывает тренажёры из `trainers/`.

## Что проверено

Проверено прохождение каждого тренажёра по правильному сценарию в браузере: все шесть доходят до финального состояния без JS‑ошибок. Это UX/функциональная проверка; полноценную методическую экспертизу доказательств и формулировок лучше сделать отдельно перед массовой публикацией.
